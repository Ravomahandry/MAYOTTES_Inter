# Carte interactive des sites

Site web cartographique statique avec **fond Esri** et **popup au survol**.

## Fonctionnalités

- fond de carte Esri
- points des sites
- popup au passage de la souris
- recherche par identifiant ou localisation
- publication facile sur GitHub Pages
